Облицовочная керамика для каркасов из оксида циркония Cercon® ceram S

1). ИНФОРМАЦИЯ ПО ПРОДУКЦИИ


Cercon® ceram S является облицовочной керамикой, которая была разработана специально и исключительно для каркасов из оксида циркония.

Были проведены многочисленные тесты облицовочной керамики Cercon® ceram S и определены показания для ее использования с каркасами из оксида циркония, выполненных из структурного материала Cercon base. С материалами, имеющими аналогичный коэффициент термического расширения (например, сплавами на основе титана) Cercon® ceram S не используется. Актуальные исследования указывают на то, что универсальность керамических масс приводит к недостаткам в эстетическом плане и/или в технических показателях. – поэтому Cercon® ceram S оптимально ориентирована на использование с оксидом циркония.

Само собой разумеется, что материаловедческие характеристики и рабочие качества Cercon® ceram S разработаны таким образом, что они превосходят настоящие стандарты облицовочных керамических масс и тем самым оптимально соответствуют требованиям по высокому качеству современных облицовочных материалов.

Преимущественные характеристики Cercon® ceram S определяются в двух словах, как:
• максимальная однородность поверхности
• оптимальное соединение с оксидом циркония

В дальнейших характеристиках качества, которые индивидуально воспринимаются врачом-стоматологом и зубным техником, таких как - передача цвета, цветовая адаптация или прозрачность - Вы убедитесь после самостоятельного апробирования Cercon® ceram S.

 

Cercon® ceram S не совместима с другими керамическими массами!
Просьба не комбинировать и не смешивать керамики с аналогичным КТР между собой.


Технические данные

Оксид циркония

В отличии от стоматологических сплавов на основе благородных металлов оксид циркония имеет меньшую теплопроводность. Для достижения желаемой конечной температуры внутри общего каркаса, материалу необходимо больше времени для поглощения температуры. Поэтому необходимо четко отслеживать выполнение рекомендаций по проведению обжига.

2). ЗУБОТЕХНИЧЕСКАЯ ИНФОРМАЦИЯ (ИНСТРУКЦИЯ ПО ПРИМЕНЕНИЮ)

Обработка каркаса

прецизионная коррекция, например в области края, выполняется с помощью алмазных инструментов

   

В любом случае шлифование осуществляется турбиной с водяным охлаждением. Для избежания перегрева материала, из которого выполнен каркас, и образования микротрещин необходимо работать без или только с незначительным давлением.

 


Для избежания напряжений в облицовочной керамике Cercon® ceram S имеющиеся углы или края каркаса округляются.

Каркасы, имеющие трещины и/или дыры, не подходят для дальнейшего использования, т.к. при этом невозможно нанесение облицовочного слоя с отсутствием дефектов и рассчитанного на длительное время использования.

 

Каркасы из оксида циркония обрабатываются в пескоструйной установке с использованием оксида алюминия (размер частиц 110-125µм, давление 3-3,5 бар). После этого каркасы очищаются с помощью пароструйного прибора (Aquaclean) или в чистой ванне ультразвуковой очистки.

 

 

Обработка керамики.


В начале монохромный белый каркас кажется технику-керамисту, длительное время работавшему с металлокерамикой, несколько непривычным. Однако после оценки эстетических результатов, которые можно достичь и короткого времени привыкания цвет каркаса будет представлять как оптимальная основа под нанесения слоев керамики.

Указания по проведению обжига

Нанесение слоя лайнера, также как и нанесение слоев дентина, массы режущего края и прозрачных масс осуществляется в соответствии с уже зарекомендовавшей себя схемой нанесения слоев по методике Ducera.

Температура обжига Cercon® ceram S лежит между 8100С и 8500С.

Специфические температурные показатели оксида циркония (низкая теплопроводность) требуют точного соблюдения указанного в таблице обжига керамики времени предварительного прогрева и шага подъема температуры.

Указанные температуры в программах обжига являются рекомендательными и должны быть индивидуально подстроены к печи для обжига керамики, которая используется в Вашей лаборатории.

Вследствие своей низкой теплопроводности каркасы из оксида циркония устанавливаются и обжигаются на специальных проволочных штифтах для обжига.

 

   


Нанесение лайнера

Лайнер Cercon® ceram S придает коронке основную цветовую насыщенность (хрому). Для этого в Вашем распоряжении различные лайнеры.

Сначала на заготовку оксида циркония Cercon base наносят тонким слоем лайнер, таким образом, индивидуализация цвета зуба начинается еще «из глубины».

Лайнер смешивается аналогично опаку с жидкостью OL и наносится тонким кроющим слоем.

Лайнер не повышает силу сцепления. Даже если отказаться от использования лайнера, сила сцепления керамики с каркасом меньше не будет.

После обжига колпачок имеет равномерно покрытую цветом, блестящую поверхность.
Если поверхность покрыта неравномерно или покрывающий слой после обжига выглядит недостаточным, можно повторить данный этап работы.

Техника работы с плечевыми массами

Плечевые массы улучшают эстетику по краю коронки.

Прежде чем шаг за шагом описать отдельные этапы работы, хотелось бы обрисовать пользу и преимущества плечевых масс:

• простые рабочие качества
• высокая стабильность
• стабильность даже при многократном обжиге
• регулируемая прозрачность
• эффект хамелеона


• отличное сцепление с каркасом
• щадящее отношение к десне

Отдельные шаги:

• срезать около 0,5 мм с края коронки, внутреннюю (в особенности у края) и наружную поверхности обработать в пескоструйной установке (в соответствии с инструкцией по применению); после этого очистить паром.

Для срезания краевой области используйте алмазы средней зернистости. Шлифование выполняется турбиной с водяным охлаждением. Для избежания напряжений края оформляются с мягким переходом.

• Обозначьте границу препарирования и покройте ее обычным образом.
• Обильно нанесите изолирующее средство для керамики Ducera-Sep. Просушите изолирующий слой воздухом и нанесите еще один слой.
• С помощью замешенной до высоковязкой консистенции плечевой массы (с добавлением жидкости SD Quick) восстанавливается переход к коронке.

Керамическое плечо просушить и приподнять (при необходимости удлинить по направлению оси).

• Теперь установить реставрацию на специальные проволочные штифты для обжига. При этом обращать внимание на то, чтобы плечевая масса не прикасалась к штифту для обжига.

Первый обжиг плеча в печи Austromat 3001

C450 T180 T180.L9 T180 V9 T060 • C850 VO T60 CO LO T2 C450

• Шлифование и подгонку плеча осуществлять без перегрева (край обрабатывать только резиновыми полировальными инструментами), если необходима коррекция, еще раз нанести плечевую массу и произвести обжиг.

Второй обжиг плеча в печи Austromat 3001

C450 T180 T180.L9 T180 V9 T060 • C850 VO T120 CO LO T2 C450


• Теперь завершайте формирование плеча. При этом необходимо обращать внимание на достаточную толщину в области перехода на слой опака.
• Очистите реставрацию и начинайте наносить следующий слой керамики.

Нанесение слоя дентина

На нанесенный лайнер накладываются восстанавливающие слои дентина и массы режущего края, таким образом, как это было описано в схеме нанесения слоев Cercon ceram S.


Для индивидуализации нанесения керамики в Вашем распоряжении имеются модификаторы и массы с эффектом опалесценции.

Глянцевый обжиг

После оформления формы и структуры поверхности коронка с нанесенной керамикой Cercon ceram S покрывается глазурью, после чего индивидуальные эффекты придаются с использованием красителей Cercon ceram S.

4). СЕРВИС

Коротко и ясно: сокращения

L лайнер, соотносящийся с определенным цветом зуба. Его пониженная способность пропускать свет обеспечивает хороший покрывающий эффект.
D дентин, соотносящийся с определенным цветом зуба, служит для восстановления керамической реставрации по образцу естественного зуба. Для лучшего контроля содержит органические цветовые пигменты, которые полностью выгорают при обжиге.
S расцветка масс режущего края, массы режущего края различных цветов, см. также объяснения по дентинам
3D хромадентин может быть смешен с различными массами или быть использованным в чистом виде, например при тонких слоях, таких как в цервикальной области, или для повышения показателя цветовой насыщенности (хрома) в области центральной окклюзии. По сравнению с дентинами (D) обладает иной насыщенностью (хромой, цветовой интенсивностью) при одинаковой прозрачности.
bambus, solaris модификаторы для изменения основного цветового тона в целях придания индивидуальности
Т, ТС, ТО прозрачные массы позволяют достигнуть дополнительные индивидуальнех эффекты прозрачности в области режущего края и, таким образом, оживляют работы при различном световом освещении
K масса для выполнения корректирующего обжига после глянцевого
Glasur глазурь, керамика мелкого помола для окончательного глазурования керамических реставраций
CD хромадентин, служит для повышения цветовой интенсивности, может быть частично смешан с дентинами или быть использован в чистом виде для изменения цветового тона.
жидкости
OL служит для смешивания всех порошкообразных опаков, придает порошку керамики густую консистенцию, делает возможным нанесение, аналогичное нанесению краски
SD служит для смешивания дентина, модификаторов, масс режущего края, прозрачных масс
SD-Form аналогична SD, при этом повышает также формоустойчивость керамических масс
SD-Color для окрашивания керамических масс и, таким образом, более эффективного нанесения слоев
Quick cлужит для замешивания дентина, модификаторов, масс режущего края, прозрачных масс, в тех случаях, когда необходимо более быстрое время просушивания. Области применения: изготовление керамических фасеток, вкладок и накладок и корректирующих обжигов (например, на контактных точках, т.к. меньшая усадка)
